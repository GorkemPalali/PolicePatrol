# Changelog

## [1.0.0] - 2025-12-19

### Added
- PostgreSQL + PostGIS database schema
- FastAPI backend with CRUD endpoints
- React + TypeScript frontend with Leaflet map
- Risk forecasting with adaptive KDE
- Grid-based risk cell generation
- Route optimization service
- ML model services (SARIMAX, Spatial, Ensemble)
- Docker Compose setup
- JSONL data import script
- Basic test suite

### Features
- **Risk Forecasting**: Spatio-temporal risk analysis with adaptive KDE
- **Route Optimization**: Risk-aware patrol route generation
- **Interactive Map**: Real-time visualization of stations, crimes, and risk heatmaps
- **ML Integration**: Time-series and spatial-temporal forecasting models

### Technical Stack
- Backend: Python 3.11, FastAPI, SQLAlchemy, PostGIS
- Frontend: React 18, TypeScript, Leaflet
- Database: PostgreSQL 15 + PostGIS 3.4
- ML: scikit-learn, statsmodels, numpy
